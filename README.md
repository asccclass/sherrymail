## Sherrymail 寄信公用程式


### Install
* get library
```
go get github.com/asccclass/sherrymail
```

* create envfile (vi envfile)
```
ailServer=smtp.abc.com.tw
MailServerPort=25
MailAccount=
MailPassword=
DocumentRoot=www/html
PORT=80
```
